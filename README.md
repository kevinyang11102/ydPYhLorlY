# 项目简介 python605

本项目是一个基于 Python 和 GIS 的机器学习模型，主要用于目标变量的空间分布预测。特别适用于多情景洪水灾害风险评估领域，能够结合 SSP 未来气候情景数据进行未来预测评估。

## 技术栈

- **编程语言：** Python
- **GIS 工具：** 集成地理信息系统相关技术
- **机器学习库：** scikit-learn, H2OAutoML, XGBoost, LightGBM 等
- **模型类型：**
  - 随机森林 (RF)
  - GBDT 梯度提升决策树
  - XGBoost
  - KNN
  - CNN
  - GNB 高斯朴素贝叶斯
  - LR 逻辑回归

## 功能模块

- **易发性评价：** 基于各种算法进行灾害易发性评估
- **承灾体灾害易损性评价：** 评估灾害对承灾体的影响程度
- **地质灾害综合风险等级评价：** 综合评价地质灾害的风险等级

## 系统角色

- 数据处理与模型构建
- 预测分析
- 结果可视化

## 运行环境

- 操作系统：Linux 或 macOS（建议），Windows（兼容）
- Python 版本：3.6 或以上
- 依赖管理：pip

## 部署步骤

1. 安装依赖库
2. 配置 GIS 环境
3. 运行数据处理脚本
4. 训练机器学习模型
5. 执行预测分析

## 目录结构

```
project/

├── data/                # 数据存储目录

│   └── ...

```

## 核心亮点

- **算法多样化：** 集成了多种机器学习算法，可针对不同场景灵活选择
- **自动化机器学习：** 采用 H2OAutoML，提高模型训练效率
- **综合风险评估：** 考虑多因素，实现地质灾害的综合风险等级评价
- **结果可视化：** 利用 GIS 工具，直观展示预测结果的空间分布
